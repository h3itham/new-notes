


#### Important Concepts. 
- Web Access Control Lists (ACLs)
	- Required entity for associating Rules / Rule Group to resources. 
- Rules 
	- Individual statements for inspection criteria. 
	-  Configure rules to perform actions. (Allow, Block, Captcha, Count)
	- Don't have an ID - Just JSON 
- Rules Group
	- Collection of rules. 
	- Create your own or buy them from AWS  Marketplace.
