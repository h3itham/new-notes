
### Project Details 
**Build Data Analytics Platform for Enterprise**
Your company is building a secure, scalable SaaS Platform that collect, process and analyzes enterprise data For business Intelligence and support multiple clients. 

| Tool / Service           | Usage in Scenario                                                            |
| ------------------------ | ---------------------------------------------------------------------------- |
| **IAM**                  | Define access control for DevOps, Developers, and Clients                    |
| **VPC**                  | Create isolated network per environment (dev, staging, prod)                 |
| **NAT**                  | Allow private instances (e.g., Composer, Data Fusion) to access the internet |
| **GCS**                  | Store raw, transformed data and Terraform state files                        |
| **KMS & Secret Manager** | Encrypt API keys, DB passwords, and service credentials                      |
| **Load Balancer**        | Expose frontend securely and distribute traffic                              |
| **IAP & Cloud Armor**    | Protect the UI with Identity-Aware Proxy and block attacks                   |
| **Cloud Run**            | Deploy microservices (auth, reporting, ingestion) as Docker containers       |
| **Cloud Functions**      | Handle event-based tasks like notifications or triggers                      |
| **Artifact Registry**    | Store Docker images used in Cloud Run                                        |
| **Composer**             | Orchestrate ETL pipelines for daily data processing                          |
| **Cloud Data Fusion**    | Build drag-and-drop pipelines for real-time client data ingestion            |
| **Git**                  | Store Terraform and Docker configs; enable version tracking                  |
| **Terraform**            | Manage and automate all infrastructure as code                               |
| **Docker**               | Containerize microservices and pipelines                                     |
