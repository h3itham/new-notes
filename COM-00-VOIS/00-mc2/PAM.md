- Privileged Access Manager for time **Temporary Access Aontrol** 
- How it works 
	- Create an Entitlement 
		1. Define who can request temporary access. 
		2. Specify what roles they can temporarily get 
		3. Set maximum duration 
	- Granting Acces
		1. A user added to the entitlement can request access
		2. if approved they receive the roles for the allowed time. 
		3. After the time expires the roles are automatically revoked. 
- There three categories of Users 
	- Requesters 
	- Approvers 
	- Notifications People
