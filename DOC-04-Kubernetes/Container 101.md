#### C Groups 
Control Groups are a Linux Kernel Feature that allow you to limit, account for and isolate the resource usage. 
**Provide the following functionalities:**
1. Resource Limiting
2. Prioritization. 
3. Accounting. 
4. Isolation
Kubernetes uses CGroups to enforce resource requests and limits. defined in pod specifications.
