#### Chapter 1 "Introduction"
- **Operating system:** Software which manage computer hardware. 
- I**nterrupts:** signals that temporarily stop the normal execution of program to give attention to a more urgent task. 
- **CPU:** is the hardware that execute the instructions. 
- **Processor:** A physical ship that contain one or more CPUs.
- **Core:** The basic computing unit of the CPU. 
- **Linked List:** Linear data structure where elements (nodes) are connected via pointers.
